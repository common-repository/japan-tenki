このプラグインは、日本の142地域の今日・明日・明後日の天気予報と予想気温をWordPressに表示することができるプラグインです。

== プラグインの説明 ==
このプラグインは、日本の142地域の今日・明日・明後日の天気予報と予想気温をWordPressに表示することができるプラグインです。
現在は、日本語のみの対応です。
将来のバージョンでは、翻訳ファイルにも対応する予定です。

= 機能 =
1.日本の142地域の今日・明日・明後日の天気予報と予想気温を表示します。
2.ショートコードで柔軟に対応することができます。

== インストールの方法 ==
1.「Japan Tenki」をプラグインディレクトリにアップロードします。
2.プラグインを有効化します。
3.ショートコードを書くことで表示されます。

== よくある質問 ==

Q&A - Weather Hacks - livedoor
http://weather.livedoor.com/weather_hacks/qa.html
をご覧ください。

= 対応している言語は何ですか？ =
現在は、日本語のみの対応です。
将来のバージョンでは、翻訳ファイルにも対応する予定です。

== スクリーンショット ==
1.日本の142地域の今日・明日・明後日の天気予報と予想気温をサイトに表示します。
2.管理画面で日本の142地域を指定する数字の番号が分かります。
