== Description ==
This plugin can display a weather forecast and expected temperature of today, tomorrow, the day after tomorrow of the Japan 142 areas.
Only Japanese supports it now.
In future versions, it is going to support the translation files.

= Functions =
1. This plugin displays a weather forecast and expected temperature of today, tomorrow, the day after tomorrow of the Japan 142 areas.
2. You can be flexible in the short code.

== Installation ==
1. Upload the `Japan Tenki` directory to the plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You write a short code and can display it.
